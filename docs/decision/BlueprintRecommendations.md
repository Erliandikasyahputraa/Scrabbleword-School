# Blueprint Architectural Recommendations

This document outlines key technical recommendations for our interactive learning platform, aligned with the MVP scope defined in `PROJECT_SCOPE.md`.

## 1. Scope Constraints (MVP Focus)
- **Included Features**: Authentication, Dashboard, Course Management, PDF uploads/viewer, Crosswords, Auto-scoring, Student history.
- **Excluded Features**: We must strictly exclude AI integrations, chat systems, gamification, and mobile apps. Any interactive templates or components outside this scope (e.g. chat modules in `shadcn-admin` or multiplayer sockets in `shadcn-table`) will not be implemented.

---

## 2. Recommended Frontend Architecture
- **Tech Stack**: React 19 + TypeScript + TailwindCSS 4 + shadcn/ui.
- **Structure**:
  - Organize folders by feature (e.g. `auth`, `dashboard`, `pdf-viewer`, `crossword`).
  - Keep the app shell layout (`AppSidebar` and `Header`) consistent, adopting the layout structures from `shadcn-admin`.

---

## 3. Recommended PDF Viewer Implementation
- **Core Library**: `pdfjs-dist` version 4+ for high-performance canvas page rendering.
- **Implementation**:
  - Build a custom React component using `useRef` to target an HTML5 `<canvas>` element for page rendering.
  - Integrate a simple viewport scale multiplier to handle zooms, and add back/next page triggers in the toolbar.
  - Implement basic virtualization to unrender pages scrolled out of view, saving browser memory.

---

## 4. Recommended Crossword Puzzle Flow
- **Fitting Engine**: Wrap the core layout algorithm (`layout_generator.js`) in a TypeScript helper (`src/utils/crossword.ts`).
- **Interactive Grid**:
  - Render crossword cells using a responsive CSS Grid:
    ```html
    <div style={{ display: 'grid', gridTemplateColumns: `repeat(${cols}, minmax(0, 1fr))` }}>
    ```
  - Highlight active cells (e.g. borders, background changes) on focus.
  - Auto-check answers client-side when the student clicks "Submit" by comparing input strings with the solution metadata generated by the fitting engine.
